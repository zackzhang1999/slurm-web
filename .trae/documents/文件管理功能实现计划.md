# 文件管理功能实现计划

## 目标
在侧边栏"统计报表"和"磁盘配额"之间添加文件管理功能，允许用户查看、删除、上传和下载自己目录下的文件。

## 功能需求

### 1. 文件浏览
- 显示用户 home 目录下的所有文件和文件夹
- 显示文件大小、修改时间
- 支持文件夹导航（进入子目录、返回上级）
- 文件图标区分（文件夹、图片、文档等）

### 2. 文件操作
- 删除文件/文件夹
- 上传文件（支持拖拽上传）
- 下载文件

### 3. 权限控制
- 普通用户只能访问自己的目录
- 管理员可以管理所有用户目录（可选）

## 技术方案

### 后端 API
- `GET /api/files?path=xxx` - 获取目录文件列表
- `POST /api/files/delete` - 删除文件
- `POST /api/files/upload` - 上传文件
- `GET /api/files/download?path=xxx` - 下载文件

### 前端页面
- 新建 `templates/file_manager.html`
- 在侧边栏添加入口
