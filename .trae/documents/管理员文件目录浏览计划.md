# 管理员文件管理目录浏览计划

## 目标
管理员登录文件管理功能时，默认显示 /home 目录，可以看到所有用户的目录。普通用户仍然只能看到自己的目录。

## 现状
- 所有用户登录后都只能访问自己的 /home/{username} 目录
- 管理员也需要切换到 /home 目录才能看到其他用户目录

## 实施方案

### 修改文件
- `app.py` - 修改文件管理 API，根据用户类型返回不同目录
- `templates/file_manager.html` - 管理员显示目录选择器

### 实现步骤

#### 1. 修改后端 API
- `/api/files` 接口增加用户类型判断
- 管理员返回 /home 目录
- 普通用户返回 /home/{username} 目录

#### 2. 修改前端页面
- 管理员显示目录选择下拉框
- 可以选择查看 /home 或 /home/{特定用户}
